project-export

Export a project.

SYNOPSIS

rerun rundeck-admin:project-export --project <> --url <$RUNDECK_URL> --user <$RUNDECK_USER> --password <$RUNDECK_PASSWORD> [--archive <>]

OPTIONS

  • –project <>: the project name.
  • –url <$RUNDECK_URL>: rundeck server url.
  • –user <$RUNDECK_USER>: the login user name.
  • –password <$RUNDECK_PASSWORD>: the login password.
  • [–archive <>]: the archive file.

README

Use project-export to export a project into an archive you can import using project-import. If no --archive option is specified one will be written to the current working directory.

Examples

Export a project called “qa1” to a file “qa1.zip”:

rerun rundeck-admin:project-export --project qa1 --archive qa1.zip \
    --url http://rundeck1:4440 \
    --user admin --password admin
