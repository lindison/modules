start

submit a job to ccm

SYNOPSIS

rerun cliqrapp:start [--runname <>] [--env_name <>] [--ClusterName <>] [--type <m1.medium>] --wait <no> [--job_list <>]: list of jobs example '169:m1.medium 168:m1.medium 167 [--fmwbaseappid <678>] [--fmwdepend <no>] [--environment <PI>]: Deployment Environment [--cloud <DTV_AWS_DevPI-us-west-2>] [--subnet <subnet-02bf4e5b>] [--vpc <vpc-82a873e7>] [--vip_pool <updawebpi-aws_POOL>] --cloud_tag <VMWEE> [--version <>] [--debug <no>] [--fmw_job_list <>] [--fmwinfo <>]: adm:wlpw [--env_pairs <>] [--appid <>]

OPTIONS

  • [–runname <>]: this is a run name and cannot be deleted defaults to appName dattime
  • [–env_name <>]: env_name to find
  • [–ClusterName <>]: ClusterName environment var
  • [–type <m1.medium>]: instance type
  • –wait : time to wait
  • [–job_list <>]: list of jobs example ‘169:m1.medium 168:m1.medium 167:m3.large’
  • [–fmwbaseappid <678>]: FMWBaseDomain base app id.
  • [–fmwdepend ]: yes if start a domin if not there
  • [–environment ]: Deployment Environment: PI,PRO,E2E
  • [–cloud <DTV_AWS_DevPI-us-west-2>]: cloud name such as DTV_AWS_DevPI-us-west-2 amazon zone
  • [–subnet <subnet-02bf4e5b>]: subnet 10.223.0 is subnet-bdb060ca subnet-02bf4e5b=10.223.2
  • [–vpc <vpc-82a873e7>]: vpc-82a873e7 is the one we have now
  • [–vip_pool <updawebpi-aws_POOL>]: if ther is a VIP POOL
  • –cloud_tag : combind clud config for each environmnet
  • [–version <>]: info on specefi version only
  • [–debug ]: instance will not stop if there are errors.
  • [–fmw_job_list <>]: joblist to pass to FMW
  • [–fmwinfo <>]: adm:wlpw:nmpw
  • [–env_pairs <>]: comma seperated pairs like v=tt,v2=ty
  • [–appid <>]: appids
